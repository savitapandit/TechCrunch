== Frequently Asked Questions ==
* How do i add logo in the header?
Logo can be uploaded via Apperence >> Header page from your Dashboard.

* How do i add Favicon icon?
Favicon icon can be uploaded via ftp to your hosting and then get the url of favicon icon and then paste it into the Theme Options page. You can visit Theme Options page via Apperence >> Theme Options.

* How to show the post thumbnail in Homepage? 
You can select Show Post Thumbnail through Theme Options page. You can change the size of Post Thumbnail through Media Page. And do not forget to set featured image for the post.

* How to show full post in Home page?
Well this feature has been added the theme since version 1.4
